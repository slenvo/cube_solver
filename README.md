# 🌌 Premium Autonomous Rubik's Cube Vision Solver

A production-grade Rubik's Cube scanning, verification, optimal sequence pathing, and interactive training dashboard environment. Built with a futuristic glassmorphic UI and real-time webcam color detection.

## ✨ Features
* **Live Camera Scanning:** Real-time multi-face color detection using webcam feeds and a guided crosshair overlay.
* **Strict Layout Verification:** Mathematical matrix validation to ensure physical cube invariants (exactly 9 instances of every core color).
* **Algorithm Optimization:** Calculates the absolute most efficient solution path and estimates execution time.
* **Premium UI/UX:** Dark-mode space aesthetic featuring deep metallic cards, frosted glassmorphism, and smooth state-machine transitions.

## 🛠️ Tech Stack
* **Frontend:** Next.js, React, TypeScript, Tailwind CSS
* **Vision & Logic:** React Webcam, Cube.js Algorithm Mapping
